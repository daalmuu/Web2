# BellaCucina

BellaCucina is a healthy Italian recipe sharing web application that allows users to explore, upload, and manage recipes in an interactive community environment.

## Features
- User registration and login
- Add, edit, and delete recipes
- Upload recipe images and videos
- Favorite , Report , Like a recipes 
- Share a comment with other
- Admin dashboard
- AJAX functionalities using jQuery
- Responsive and modern UI design

## Technologies Used
- PHP
- MySQL
- HTML5
- CSS
- JavaScript
- jQuery & AJAX

## Database
Import the provided SQL file into MySQL before running the project locally.
 
## Local Setup
1. Place the project folder inside htdocs
2. Start Apache and MySQL using MAMP/XAMPP
3. Import the database file
4. Update DB.php connection settings
5. Open the project in the browser

For MAMP
- $conn = new mysqli("localhost", "root", "root", "bellacucina_db", 8889);
  
For XAMPP
- $conn = new mysqli("localhost", "root", "", "bellacucina_db", 3306);
